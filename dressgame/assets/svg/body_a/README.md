# Body A SVG Asset Pack

This folder contains hand-made SVG assets with a shared canvas and anchor system.

- Canvas: `1024x1536`
- Body type: `body_a`
- Layer order: `base -> bottom -> top -> shoes -> hair -> accessory`

## Included Parts

- Base: 1
- Hair: 3
- Top: 3
- Bottom: 3
- Shoes: 2
- Accessory: 3

## Notes

- Every file uses the same viewBox for stable overlay alignment.
- Files are designed for browser-based image layering (`<img>` stack or canvas).
- The default set is described in `items.json`.
